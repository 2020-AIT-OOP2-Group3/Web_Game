## 仕様
- ポイント制Webゲーム
  - 新規登録（またはログイン）をし、初期持ちポイントをうけとる。ID・パスワード入力が必要とする。
  - それぞれ、ゲームを行いポイントを集める。ゲームをやるごとに賭け金としてポイントを支払う。
  - クイズゲーム… ジャンルとしては脳トレ 
  - ババ抜きゲーム… ３人CPUとプレイヤーで４人対戦
